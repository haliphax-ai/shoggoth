# `@shoggoth/cli`

Operator `shoggoth` CLI: control-socket client, skills listing, HITL, retention, MCP helpers, and session compaction.

- **Source:** `src/`
- **Tests:** `test/` (`npm run test`)
- **Typecheck:** `npm run typecheck` → `tsc --noEmit`. Bin `run-cli.mjs` spawns `node --import tsx/esm src/cli.ts`.
